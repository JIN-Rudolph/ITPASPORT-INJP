🌳 木構造 (Tree Structure)

基本用語

  • root（ルート）：いちばん上のノード
  • parent / child（親 / 子）：つながりの関係
  • leaf（葉）：子がないノード
  • depth（深さ）：root からの段数
root
├─ child A
│ ├─ leaf A1
│ └─ leaf A2
└─ child B

どんな時につかう？

  • フォルダ・ファイルの階層、Web の DOM 構造
  • 組織図、家系図、分類表（taxonomy）
